Project Pulmonary Website (static)

Quick start:
- Unzip
- Open index.html in your browser

Enable real form submissions:
- Add a JSON endpoint to each form:
  data-endpoint="https://your-endpoint"
Common options:
- Formspree (with a compatible endpoint)
- Netlify Forms (uses HTML form attributes)
- Custom serverless function (Netlify/Vercel)

Image placeholders:
Replace files in assets/img/ with real images:
- og-image.jpg (1200x630)
- hero.jpg
- gallery-1.jpg, gallery-2.jpg, gallery-3.jpg
